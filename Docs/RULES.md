# Quy Tắc Làm Việc Dự Án

- Bảo mật và an toàn: không commit secrets, kiểm tra input.
- Code rõ ràng, dễ bảo trì; mỗi hàm làm một việc.
- Không sửa file dependency thủ công; dùng package manager.
- Test: unit/integration/E2E tối thiểu 70% cho logic quan trọng.
- Ghi CHANGELOG cho mọi thay đổi; cập nhật TASKS khi tiến độ đổi.
